Quick Start

# Install pip install cylinderwake3900 # Python from cylinderwake import CylinderWake3900 # Eulerian fields (auto-download) ds = CylinderWake3900("eulerian", "near") sample = ds[0] print(sample["velocity"].shape) # Lagrangian particles ds = CylinderWake3900("lagrangian", "near") # AI agent metadata from cylinderwake.api import get_dataset_card_json print(get_dataset_card_json())

Benchmark Tasks

  • Temporal Forecasting
    Predict next velocity snapshot from history
  • Super-Resolution
    Reconstruct high-res from coarsened fields (4x, 8x, 16x)
  • Sparse Reconstruction (Unique!)
    Lagrangian particles to Eulerian field — paired data
  • Lagrangian Prediction
    Predict particle trajectories from Eulerian field
